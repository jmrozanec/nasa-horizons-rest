NASA NEO
  • Accessibility of NEO data: REST APIs to feed our Skynet.
    Problems identified:
    - Most data was not available in a machine readable / accessible format
    Our contribution:
    - Extracted data in HTML tables into a database.
    - Interfaced NASA's HORIZONS System telnet client, to provide functionality missing at web interface and make orbital information widely accessible.
    - Exposed this information with a REST client

Developed as part of the 2016 NASA Space App Challenge
